\doxysubsection{Detailed Description}
Core of the session system. Manages sessions. 

\mbox{\hyperlink{class_session_manager}{Session\+Manager}} enables session creation, deletion, closing, opening and saving. It is also responsible for updating \mbox{\hyperlink{class_main_window}{Main\+Window}} with session information.
