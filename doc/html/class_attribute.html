Detailed Description

The Attribute class.

Represents one attribute of NPC character, stores abilities of this attribute, can reroll abilities lvl

Uses Abilities class to contain abilities, also uses it to roll abilities lvl points Uses attributes enum with names to store it's name Uses Dice class to store dice type
