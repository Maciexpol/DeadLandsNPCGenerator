\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_attribute}{Attribute}} class. 

Represents one attribute of NPC character, stores abilities of this attribute, can reroll abilities lvl

Uses \mbox{\hyperlink{class_abilities}{Abilities}} class to contain abilities, also uses it to roll abilities lvl points Uses attributes enum with names to store it\textquotesingle{}s name Uses \mbox{\hyperlink{class_dice}{Dice}} class to store dice type
